## 红线速查

以下违反任一条 = 方案打回 / CR 不通过：

### 架构红线
1. 禁止循环依赖（模块间 A → B → A）
2. 领域层不得依赖框架/ORM
3. 禁止跨层调用（表现层直接调基础设施层）
4. 敏感信息（密钥/密码）禁止硬编码

### 代码红线
1. 禁止 `print()` → 用 `logging`
2. 禁止裸 SQL 拼接 → 参数化查询
3. 密钥/Token/密码 禁止硬编码 → 环境变量
4. 禁止遗留调试代码（print/console.log/注释代码块）

### 流程红线
1. `main` 分支禁止直接 push
2. Conventional Commits 格式
3. 跳过可视化验证 = 功能点不通过
4. PR 至少 1 人 Review
