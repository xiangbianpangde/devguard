# 汇报 (docs/reports/)

> 对应 `conventions/06-documentation_文档规范.md`、模板见 `docs/templates/汇报模板.md`

存放从 `汇报模板.md` 生成的各档**进度汇报**。汇报是 `STATUS.md` / `worklogs/` 的对外提炼，不重复记录事实。

## 命名约定

| 档位 | 文件名 | 触发 |
|------|--------|------|
| 功能点完成 | `功能点{编号}-{简短名}.md` | 单功能点交付时 |
| 日报 | `日报-YYYY-MM-DD.md` | 每日（可选） |
| 周报 | `周报-YYYY-WNN.md` | 每周 |
| 阶段/里程碑 | `阶段汇报-{阶段名}.md` | 阶段结束 / 里程碑 |

> 周期汇报（日/周报）累积为阶段汇报的素材；阶段汇报对内看进度、对上看里程碑，合并为一份。
