# 汇报 (docs/reports/)

> **👤 人类参考** | 对应规范：`conventions/06-documentation_文档规范.md`
> 模板：`docs/templates/汇报模板.md`
> 更新: 2026-07-11

**本目录主要存放收束报告。** 汇报是 `STATUS.md` / `worklogs/` 的对外提炼，不重复记录事实。

## 落盘的汇报

| 档位 | 文件名 | 触发时机 |
|------|--------|---------|
| 收束报告 | `收束报告-v{版本号}.md` | 每 3 个功能点 / 版本号变更 |

> **功能点级汇报默认对话内联、不落盘**（见 `conventions/ai-workflow_AI协作开发流程/07-汇报.md`）；仅人要求留档时才在此生成 `功能点{编号}-*.md`。历史遗留的功能点汇报文档保留不动。
> AI 填数据，人做判断（详见 `docs/templates/汇报模板.md` 的"AI 填写指引"）。
