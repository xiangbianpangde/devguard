## 二、阶段 V0.x 起步：四件套同步建立（2026-05-26）

### 目标

让 devguard 在 V0.x 起步阶段就有完整的"项目门面"：人/AI/数据/可视化四件齐全。

### 实现路径

```
仓库初始化
    │
    ├── README.md          # 人类入口（项目说明 + 5 步快速开始）
    ├── CLAUDE.md          # AI 入口（自动加载的项目上下文）
    ├── STATUS.md          # 进度数据源（表格化功能点状态）
    └── dashboard.html     # 可视化面板（解析 STATUS.md）
```

### 改动

| 文件 | 改什么 | 状态 |
|------|--------|:---:|
| README.md | 新建：人类导航 + 5 步快速开始 + 目录树 | ✅ |
| CLAUDE.md | 新建：AI 上下文 + 目录索引 + 行为规则 | ✅ |
| STATUS.md | 新建：当前进度 + 收束节点历史 + 详细功能点列表 | ✅ |
| dashboard.html | 新建：纯静态 + CSP 头 + 学术风 | ✅ |

### 关键决策

- **AI 入口命名**用 `CLAUDE.md` 而非 `AI.md` —— 沿用 Claude Code 工具约定，0 配置即自动加载
- **dashboard 用纯静态 HTML** —— 引入 React/Vue 不必要的复杂度；fetch 读 STATUS.md 即可
- **README 顶部标 `👤`** —— 视觉信号提醒人类这是给他们看的
- **CLAUDE 顶部标 `⚠️`** —— 视觉信号提醒 AI 不要把它当 README 读

### 阻塞

无。
