## 四、阶段 V0.x 重构：v2.0 / v3.0（2026-05-27 ~ 2026-05-28）

### 目标 1：01-06 规范升 v2.0

- 把所有规范结构统一为 5 段式（红线 / 配置 / 决策 / 反模式 / 检查清单）
- 每条红线必须配 L1 自动检测
- 跨规范去重（同一主题只在一处权威）
- 调用图主题统一收拢到 08

### 目标 2：08 升 v3.0 双图谱

替换 v1.0 的"图数据库 + Cypher + Mermaid"方案 → **CodeGraph（AI 用）+ Understand-Anything（人用）双图谱范式**。

### 改动

| 模块 | 改什么 | 状态 |
|------|--------|:---:|
| 01-06 + 08 规范 | 全部按 5 段式重写 | ✅ |
| docs/specs/ | 7 份 BDD realign | ✅ |
| src/<维度>/ | 5 个落地配置补齐 | ✅ |
| 根目录 + conventions | 三件 CLAUDE 文件（导航分离） | ✅ |
| docs/templates/devguard/ | code-understanding/ 示例更新 | ✅ |

### ADR 决策

- ADR 0001-0004：双图谱方案选型
- ADR 0005：长文拆目录
- ADR 0006：ADR 仅收束节点产出
