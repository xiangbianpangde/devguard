## 核心理念

1. **简洁优先** — 不过度设计，解决当前问题
2. **可维护性** — 代码是写给人看的，顺便给机器运行
3. **一致性** — 同一项目内风格统一
4. **渐进式** — 从最小可行规范开始，逐步完善
