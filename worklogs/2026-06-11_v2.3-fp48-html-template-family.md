## 关键决策

- **契约锚点 = `<meta name="doc-template" content="…">`**：产出物从模板复制时携带类型声明，校验按声明走（结构而非逐字、不猜文件名）；删声明=退化为 WARN，但模板族自身有 L4 自举测试兜底。
- **视觉基准 = AgentHub dashboard（用户 2026-06-11 指定）**：初版我做成米黄面板+衬线 700 学术风，用户打断指认 `字节比赛/AgentHub/dashboard.html`——Claude 官网风：暖米底 `#f5f4ee` + **白卡片** + 黏土 `#cc785c` + 衬线 weight 500 + tint 徽章 + 左缘色条卡。四套模板整体重写为该令牌系，素材库 §三 沉淀为唯一权威。
- **untyped 仅 WARN**：22 个存量 HTML（M1-M8 设计文件夹 + 历史报告）不强迁，避免误拦→滥豁免（设计 §1.4）。

## 遇到的问题

- **真机验证抓到甘特图语法错**：实施设计模板的 gantt 用 `YYYY-MM-DD` 占位日期，mermaid 解析失败渲染出 Syntax error——改真实示例日期 + 注释提示替换。占位符规约对"需被引擎解析的值"不适用，素材库用法表已写明。
- Playwright 对 `file://` 禁访 → 走 `scripts/start_server.py` 本地 HTTP（8080）真机渲染。
- 误判教训：用正则在 SVG 全文搜 "error" 会命中 mermaid 注入 CSS 的 `.error-icon` 规则，改查 `.error-text` 元素与 `g.node`/`rect.task` 实际图元。
