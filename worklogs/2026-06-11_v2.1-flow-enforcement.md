## 关键决策

| 决策 | 原因 | 影响 |
|------|------|------|
| commit-msg 阶段放 4 个钩子（非 pre-commit） | 豁免标记 `[skip-*]` 在 commit message 中，commit-msg 阶段才能读到 | 钩子语法保持一致性，但需确保 pre-commit 框架 install 了 commit-msg 类型 |
| MD022/MD032 标记延后 batch-2 | 违规量极大（255+339），auto-fix 改动面巨大，需分批 | 启用的 5 条规则（MD001/012/031/047/058）已覆盖最关键的格式结构 |
| 跨文件计数 WARN 不拦截 | 方案风险章节明确"先短期豁免历史差异" | 开发清单 46 vs STATUS 57 → WARN 正确产出，待人工对齐 |
| file_placement grade=warning | 决策树是启发式规则（不是硬分类器），人工判断优先 | 根目录散文件硬拦，未知目录仅 WARN |
| 收束产物检查对未签核节点仅 WARN | 收束未完成 = ADR/报告尚在产出中 | 已签核节点（v0.1）正确通过——6 ADR + 15 报告 |
